Face Recognition

Purpose

To identify whom a person is from a picture.

Design Challenges

Principal Component Analysis

Principal Component Analysis is used to compute the top 25 eigenfaces which is used to reconstruct a given input image to be compared with the reconstructed faces of the training set.

The Face Recognition program works well as shown in the Report.
