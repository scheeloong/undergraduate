Rube's Goldberg Toy Box

Purpose

The purpose of this project is to close a toy box using a Rube’s Goldberg Machine. This motivates little children to keep their toys in a toy box as it makes them keep their toys in a box just so they can play the Rube’s Goldberg Machine. The Rube’s Goldberg Machine has to has at least 20 different steps and complete all of them in a single run.

Goal

Our goal was to completely built a Rube’s Goldberg Machine that would run twenty different steps sequentially in a single run on the presentation day.
