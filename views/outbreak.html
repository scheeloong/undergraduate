Outbreak

Purpose

Outbreak of viruses occurs rapidly. It will be useful to simulate how an outbreak travels. Given a list of cities, their distances, and the initial city that got infected, this project list the order in which the cities get infected.

Design Challenges

Djikstra's Algorithm

We used the Djikstra's Algorithm to calculate which cities will get infected next.
