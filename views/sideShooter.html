Side Shooter

Purpose

This project is to program a horizontal 2D game.

Design Challenges

Collision Detection

We used rectangular bounding boxes for all our sprites and implement collision detection on rectangular bounding boxes.

Bullet Projectile

We made the bullet fly a horizontal path using basic physics.

Generation of Obstacles

We generated the obstacles at random locations on the other side of the screen.

Result

The sideShooter can be played in 2 modes as shown in Video 1 and Video 2. In the first mode, both players team up to hit as many obstacles as possible. In the second mode, both players battle each other and try to destroy each other.
