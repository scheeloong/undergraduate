Sound Note

Purpose

To create a free Ear Training website for music students.

Design Challenges

Password Encryption

We need to encrypt password. We used the bcrypt library to do so.

Result

The Ear Training website works perfectly as shown in Video 1.

Video 1: A Demonstration of the Sound Note Ear Training Website.

Report

Code

Technologies

Javascript - Implement Perfect Pitch, Relative Pitch, Chords ear training games

Web Audio - Used to output sounds for the various ear training games

ExpressJs - To dynamically display pages on the website

MongoDb - To store user information such as login, encrypted password, and current scores.

HTML + CSS + Twitter Bootstrap - Layout and outlook of the webpages
