Sudoku

Purpose

This program solves a Sudoku problem using Constraint Satisfaction Problem using the All-Different constraint by pruning the variables until a solution is found. It stops if merely using the propagation algorithm does not completely solve the Sudoku boards and leave the final solution as the possible domain values for each cell as shown in Video 1.

Design Challenges

All-Different Constraint

I used the All-Difference constraint to prune the variables. There are 3 different types of all different constraints, i) Each row must be all-different in its values ii) Each column must be all-different in its values iii) Each subcell must be all-different in its values

Binary Constraint

I also implemented the binary not-equal constraint as a comparison for the all-different constraint. Obviously, the All-Different constraint performs much better than the binary not-equal constraint.

Result

The Sudoku Solver works perfectly as shown in Video 1
