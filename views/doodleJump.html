Doodle Jump

Purpose

This project is to program a vertical 2D game.

Design Challenges

Collision Detection

We used rectangular bounding boxes for all our sprites and implement collision detection on rectangular bounding boxes.

Generation of Ledges

We generated the obstacles at random locations on the screen making sure that each ledge is generated such that it is possible for the player to jump to it in time. In other words, we made sure we do not generate any impossible ledges.

Gravity

We implemented gravity so that the jumping sprite will land and jump similar to the motion of an actual person.
