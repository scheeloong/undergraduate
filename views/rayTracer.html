Ray Tracer

Purpose

To build an entire Ray Tracer application from scratch using C++.

Design Challenges

Long Rendering Times

It takes more than a day to render the entire image on a CPU. Therefore, I have divided up the program to output different effects separately so that each rendering time will be less than a minute. I also set up parameters so a user can easily change the parameter values to have a trade off between visual appearance and rendering time.

Result

The Ray Tracer program works perfectly as shown.
